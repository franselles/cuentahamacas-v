# 🏖️ CuentaHamacas V

Sistema de gestión de inventario y control de stock en tiempo real para sectores de playa. Diseñado para facilitar el reporte de incidencias (rotas, retiradas, nuevas) de hamacas y sombrillas de forma ágil, robusta y optimizada para dispositivos móviles.

## ✨ Características

- **Gestión por Sectores:** Visualización jerárquica de la capacidad total de cada zona de la playa.
- **Reporte de Incidencias:** Formulario táctil optimizado para operarios en campo:
  - ✅ **Repuestas:** Registro de nuevas unidades que se suman al stock operativo.
  - 📦 **Retiradas:** Control de material que sale de la playa hacia almacén.
  - ❌ **Rotas:** Gestión de bajas por daños para reposición inmediata.
- **Interfaz Moderna:** Desarrollada con **Nuxt UI** y **Tailwind CSS**, con soporte nativo para **Modo Oscuro**.
- **UX Fluida:** Uso de *Skeletons* para estados de carga y notificaciones *Toast* para feedback instantáneo del servidor.

## 🚀 Stack Tecnológico

- **Frontend & Backend:** [Nuxt 4](https://nuxt.com/) (Estructura de directorios v4).
- **UI Kit:** [Nuxt UI](https://ui.nuxt.com/) basado en Radix Vue y Tailwind.
- **Base de Datos:** [MariaDB](https://mariadb.org/) para persistencia de datos relacionales.
- **Runtime:** Node.js / Nitro.
